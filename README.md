# flutter_crosspay

![License](https://img.shields.io/badge/License-MIT-blue.svg)

Cross platform flutter in-app-purchase library that supports Android, iOS, web, Linux, Windows, macOS

### Installation

We still haven't published our packages to pub.dev so we've to use git

```yaml
dependencies:
  flutter_crosspay:
    git:
      url: https://github.com/maxint-app/crospay.git
      path: dart/flutter_crosspay
```
